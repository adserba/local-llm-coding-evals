# Arcade Flight Benchmark App

A tiny intentionally imperfect app for model evaluation.

## Files

- `index.html`
- `styles.css`
- `app.js`

## Known built-in issues

This app intentionally contains a few flaws so models can be tested on realistic edits:

- camera follow is jerky
- speed label does not always refresh
- keys can stick after focus loss
- pause system is missing
- code has some duplication

## Controls

- `W` / `S` — throttle up / down
- `A` / `D` — turn left / right
- `ArrowUp` / `ArrowDown` — pitch up / down
